Tasador es una prueba de concepto o "demo" creado para WeekendCode 2021

Puedes usar esta app para estimar el precio de venta de una propiedad en la Ciudad de Buenos Aires.

El modelo usa características básicas de una propiedad (numero de ambientes, baños, superficie, etc.) para realizar esta predicción.
